# BNGS; Application Layer Protocol
## Simple Bulletin Board

This was developed solely on Windows 10.
Tested in Powershell.
Python Version: 3.9.9.

Running Instructions:
(Please always run the server first!)
On Windows:
  - Open Powershell
  - Navigate to BNGS-Protocol/
  
  - FOR SERVER:
      - Run the following: ./server/server.py
  - FOR CLIENT:
      - Run the following: ./client/client.py
  
  - These commands should pop out a new window for the process.

On Linux:
  - Open a Terminal
  - Navigate to BNGS-Protocol/

  - FOR SERVER:
      - Run the following: python3 server/server.py
  - FOR CLIENT:
      - Run the following: python3 client/client.py
